# Demo Script

Use this flow when showing the project to a recruiter:

1. Open the dashboard.
2. Point out the watchlist topics.
3. Click `Refresh live`.
4. Explain that the app ingests public RSS headlines.
5. Show the topic cards and the overall sentiment score.
6. Open the JSON export.
7. Show the health panel and mention the SQLite store.
8. Close by explaining that the app is split into ingestion, scoring, persistence, and presentation layers.

### One-line summary

> This is a standalone market sentiment analytics app that turns public headlines into a persistent dashboard and API.
